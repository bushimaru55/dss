# Phase 2 Scope

## 目的
Phase 2 では以下を実装対象とする。

- datasets: アップロードされた CSV / XLSX の管理
- profiling: 行数、列数、型推定、null率、サンプル値抽出
- semantic_mapping: 列意味推定とユーザー修正

## 対象外
- suggestions の本実装
- execution plan の本実装
- AI分析実行の本実装

## 完了条件
- Dataset 作成ができる
- Sheet 選択ができる
- Profile 生成ができる
- Semantic label の保存と更新ができる
