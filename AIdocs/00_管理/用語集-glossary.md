# 用語集（表インテリジェンス）スタブ

本ファイルは SPEC-TI 群で使う **内部用語** と **利用者向け表現** を集約する。各仕様書ではここへの参照を優先し、二重定義を避ける。

| 内部用語 | 利用者向け（例） | 定義先（予定） |
|----------|------------------|----------------|
| grain | 1行あたりの意味／1件分のデータ | SPEC-TI-004 |
| table taxonomy | 表の種類 | SPEC-TI-009 |
| heading tree | 見出しの階層 | SPEC-TI-010 |
| confidence score | 自動判定の確かさ | SPEC-TI-011 |

追記ルール: 新語は Draft SPEC で初出する場合も、Approved 前に本表へ1行追加する。
